# 🚀 Сборка Windows .exe

Эта папка содержит всё необходимое для сборки .exe файла на Windows.

## 📁 Содержимое

| Файл | Назначение |
|------|-----------|
| `BUILD_WINDOWS.bat` | Автоматическая сборка .exe (один клик) |
| `QUICK_START.bat` | Быстрый запуск без сборки |
| `LAUNCHER.bat` | Альтернативный лаунчер |
| `icon.ico` | Иконка для Windows |
| `icon.svg` | Оригинальная SVG иконка |
| `WINDOWS_EXE_GUIDE.md` | Подробная инструкция |

## ⚡ Быстрый старт

### Способ 1: Сборка .exe
1. Скопируйте **эту папку** на Windows
2. Кликните **`BUILD_WINDOWS.bat`**
3. Готовый .exe появится в `dist\Game Library.exe`

### Способ 2: Запуск без сборки
1. Скопируйте **эту папку + всю папку `src` и `database`** на Windows
2. Кликните **`QUICK_START.bat`**

## 📋 Требования на Windows

- ✅ Python 3.8+
- ✅ PostgreSQL 12+
- ✅ Интернет (для установки зависимостей)
