# 🐧 Game Library для Linux

Один исполняемый файл — работает на любом Linux без установки Python!

## ⚡ Быстрый запуск

```bash
# Просто запустите:
./GameLibrary

# Или через терминал:
chmod +x GameLibrary
./GameLibrary
```

## 📋 Требования

- ✅ PostgreSQL (запущенный)
- ✅ x86_64 архитектура (обычный ПК)

**Не нужно:**
- ❌ Python
- ❌ pip
- ❌ Зависимости

Всё встроено в один файл!

## 🔨 Пересборка

Если нужно собрать заново:

```bash
./build_linux.sh
```

Скрипт сам установит PyInstaller и соберёт исполняемый файл.

## 📦 Распространение

Просто скопируйте файл `GameLibrary` — всё работает из коробки!

```bash
# На другой машине:
scp GameLibrary user@other-pc:/home/user/

# Запуск:
./GameLibrary
```
